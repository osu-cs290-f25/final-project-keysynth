# KeySynth
A browser-based playable polyphonic subtractive synthesizer with saveable presets.

## Features
- Adjustable settings
  - Volume
  - Gain envelope attack
  - Gain envelope release
  - Low-pass filter frequency
  - Detune amount for two voices
  - Waveform (Sine, triangle, saw, square)
- Savable Presets
- Over 3 octaves of playable keys in the key of C major. (C3-E5)

## Install
- Dependencies: Node, Express, a computer
- `npm start`
- Access through localhost:8500 in ur fav browser
